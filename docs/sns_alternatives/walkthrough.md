# Walkthrough - SNS情報代替実装

## 実装内容

Twitterの埋め込みウィジェットが機能しない問題に対し、**3つの代替情報源**を実装しました。

### 1. Yahoo!ニュース災害情報 (iframe)
#### [MODIFY] [index.html](file:///Users/saki/lab/anken/disaster-prevention-system/index.html)

```html
<iframe src="https://news.yahoo.co.jp/search?p=地震+災害&ei=UTF-8">
```

- Yahoo!ニュースの災害検索結果ページを埋め込み
- 確実に動作し、最新ニュースが表示される
- 500pxの高さで見やすく配置

### 2. 公式情報源リンク集

**実装したリンク**:
- **NHK 災害情報**: `https://www3.nhk.or.jp/news/saigai/`
  - 最新の災害ニュース・警報
- **気象庁 防災情報**: `https://www.jma.go.jp/bosai/`
  - 地震・気象警報の公式情報
- **Yahoo!災害情報**: `https://crisis.yahoo.co.jp/`
  - リアルタイム災害マップ

各リンクは：
- カード形式で視覚的に見やすい
- アイコン付きで識別しやすい
- ホバー時にハイライト

### 3. 最新地震情報リスト (JMA API)
#### [MODIFY] [js/app.js](file:///Users/saki/lab/anken/disaster-prevention-system/js/app.js)

**新メソッド**: `renderJMAEarthquakeList()`

- JMA APIから取得した地震データを一覧表示
- 最大10件の最新地震を表示
- クリックすると地図にズーム（`flyTo`）して該当位置を表示
- 震度・マグニチュード・発生時刻を表示

**データがない場合**:
```
現在、地震情報はありません。
```

### 4. SNS検索リンク

Twitterの直接検索へのリンク：
- `#地震` のリアルタイム検索
- `#災害` のリアルタイム検索
- `@nhk_news` 公式アカウント

新しいタブで開くため、元のページに戻りやすい。
